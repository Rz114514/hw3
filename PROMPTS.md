# 提示词（Prompts）

用于本项目（AI Coding IDE + Vercel）的提示词整理如下：

## 1) 任务指令

- 在空文件夹中，用 AI Coding 的 IDE 创建个人网站，并使用 Vercel 部署；提交源代码、运行截图与提示词。

## 2) 站点要求

- 纯静态（HTML/CSS/JS），无需构建；包含 Hero/关于/项目/联系；支持深浅色主题；响应式；提供 `vercel.json` 与 `README.md`、`PROMPTS.md`。

## 3) 个性化信息

- 可在 `index.html` 中替换姓名、邮箱、GitHub、头像链接。

## 4) 交互与细节

- 主题偏好存储在 `localStorage`；联系表单仅本地演示；样式清爽。

## 5) 交付物

- 源代码（完整目录）；运行截图（本地、Vercel 线上、Vercel 控制台）；提示词（本文件）。
